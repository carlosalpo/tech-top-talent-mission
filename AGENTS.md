# Tech Top Talent Mission Dashboard

## Objetivo
Construir una aplicación web interactiva para la academia Tech Top Talent, enfocada en entrenamientos de arquitectura tecnológica, networking, datacenter, nube híbrida, continuidad, integración, automatización y operación.

## Estilo visual
- Tema sci-fi / mission control.
- Fondo espacial con planeta, estrellas, naves y estética futurista.
- Colores principales: negro profundo, azul/cyan, ámbar/dorado, violeta.
- No usar nombres, logos ni referencias directas a franquicias protegidas.
- Debe sentirse profesional, no infantil.

## Stack
- React
- TypeScript
- Vite
- Framer Motion
- Assets en /public/images

## Reglas
- No usar rutas /mnt/data.
- Imágenes permanentes van en /public/images.
- Código va en /src.
- Mantener componentes separados.
- Evitar dependencias innecesarias.
- Usar SVG inline para íconos si evita problemas de paquetes.
- Cada cambio importante debe poder desplegarse en Vercel.

## Misiones
1. Infraestructura de DC Enterprise
2. Alta Disponibilidad, Continuidad y Disaster Recovery
3. Nube Híbrida e Interconexión con Nube Pública
4. Integración, Automatización y Operación

## UX esperada
- Dashboard inicial con cuatro misiones.
- Hover animations en tarjetas.
- Futuras pantallas de briefing por misión.
- Progreso guardable localmente.
- Experiencia tipo “misión”, no clase académica.
